# Netoi Index

Static GitHub Pages site for `www.netoi7.me`.

## Editing the library

- Add or edit entries in `data/resources.js`
- The UI is rendered by `app.js`
- Styling lives in `styles.css`

## Structure

- `index.html`: shell and semantic layout
- `data/resources.js`: categories, collections, and resource data
- `app.js`: search, filters, anchors, rendering, and interactions
- `styles.css`: dark editorial visual system
